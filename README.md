# Polling & Chat Application
A simple polling and real-time chat application.

## Development
This project was generated with Angular CLI.


## Tech Stack
- Angular
- Node.js
- WebSocket / Socket.IO

## Setup
npm install
npm start

## Features
- Create and vote on polls
- Live poll result updates
- Real-time chat
- Mobile-first UI
- JWT Authentication

## Notes
- UI is mobile-first; desktop responsiveness is minimal
- Basic validation and error handling implemented
